This repository contains my work for learning Pandas via various tutorials. 

The folder ```testing_pandas_install``` contains initial testing of Pandas installation via pip (```pip install Pandas```). This was done viewing two web tutorials: 
* https://www.learnpython.org/en/Pandas_Basics
* https://www.shanelynn.ie/select-pandas-dataframe-rows-and-columns-using-iloc-loc-and-ix/

The folder ```data_sci_yt_tutorial``` contains work from YouTube tutorial on introduction to Pandas: https://www.youtube.com/watch?v=vmEHCJofslg
* My code for this is contained in a single file ```data_science_yt_tutorial```
* The data for this work was taken from here: https://github.com/KeithGalli/pandas

The folder ```real_tasks_yt_tutorial``` contains work from YouTube tutorial on apply Pandas in a sales analysis application: https://www.youtube.com/watch?v=vmEHCJofslg
* My code for this is contained over seven files: a data merging file, a data cleaning file, and 5 solution files corresponding to 5 presented analysis questions (found in YouTube video description). 
* The data for this work was taken from here: https://github.com/KeithGalli/Pandas-Data-Science-Tasks
